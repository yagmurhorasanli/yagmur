# 💕 Love Website — Seni Seviyorum

Sevgilim için hazırlanmış özel bir website.  
Doğum günü ve Sevgililer Günü için aşk dolu bir sayfa.

## 🌟 Özellikler

- 💫 Yüzen kalp partikülleri (canvas animasyonu)
- 🎂 Doğum gününe sayaç
- 🕯️ Üfle-mumları-söndür etkileşimi + konfeti
- 💝 Sevgililer Günü kalp nabız efekti
- 📖 Otomatik kayan sonsuz sevgi nedenleri
- 💌 Açılabilir zarf + aşk mektubu
- ✨ Tıklama ile kalp emoji efekti
- 📱 Tam mobil/responsive tasarım

## 🎂 Doğum Günü Tarihini Ayarla

`script.js` dosyasını açıp şu satırları bul ve sevgilinin doğum gününü gir:

```js
const BIRTHDAY_MONTH = 4; // Ay (1-12)
const BIRTHDAY_DAY   = 7; // Gün
```

## 🚀 GitHub Pages'e Yükleme

```bash
# 1. Klasörü git repo yap
git init
git add .
git commit -m "💕 aşk sitesi"

# 2. GitHub'a yükle
git remote add origin https://github.com/KULLANICI_ADIN/REPO_ADIN.git
git branch -M main
git push -u origin main

# 3. GitHub Pages'i aç:
# GitHub → Repo → Settings → Pages → Source: main / root → Save
```

Yayın adresi: `https://KULLANICI_ADIN.github.io/REPO_ADIN`
